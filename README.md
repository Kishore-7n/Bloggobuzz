# Bloggobuzz

Bloggobuzz is a independent platfrom where users can create and post the blogs freely.
Bloggobuzz has a well designed user interface and a smooth user experience.

# TechStack 

Frontend

Reactjs,

Material ui

bootstarp
  
Back end

  Nodejs
  
Database

  Mongodb
